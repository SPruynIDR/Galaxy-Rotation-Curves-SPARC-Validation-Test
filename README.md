# Galaxy Rotation Curves: SPARC Validation Test

Disk galaxy rotation curves are normally explained by particle dark matter.
This is runnable proof that a single fixed equation — zero per-galaxy free
parameters — matches 171 real SPARC galaxies better than MOND and beats
Newtonian gravity (baryons only) alone. No halo. No tuning.
