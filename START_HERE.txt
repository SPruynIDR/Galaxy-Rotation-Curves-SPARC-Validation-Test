Upload every file in this folder to the same GitHub repository folder.

To run locally:
python -m pip install -r requirements.txt
python run_full_head_to_head_benchmark.py

The script uses the included raw SPARC catalog and mass-model tables.
It fits two NFW parameters independently for each of 171 galaxies,
writes all best-fit parameters and point predictions, and calculates
NFW and boundary-relation AIC and BIC values.
